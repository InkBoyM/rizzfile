How to open .rizz files

Lets try to make a .png file more fun! And only people with the scripts know how to open it!

Windows
MacOS
1. Open notepad and paste this in 
@echo off
start chrome "%~1"

Save this as open_rizz.bat (replace chrome with msedge or firefox if needed, or use start for default browser)

2. Associate .rizz Files

- Right-click any .rizz file → Open with → Choose another app
- Select "Look for another app on this PC"
- Navigate to your open_rizz.bat file and select it
- Check "Always use this app to open .rizz files"

1. Create the Opener Script (Heads up: idk if it works since i use windows so i just asked chatgpt lol :/) 
#!/bin/bash
open -a "Google Chrome" "$1"

Open Terminal and save this as: ~/open_rizz.sh

2. Make the Script Work 
chmod +x ~/open_rizz.sh
